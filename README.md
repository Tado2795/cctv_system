# cctv_system

OpenCV와 텔레그램을 이용한 자동 모션 감지 시스템

## 필요사항

  텔레그램 봇 api token, Bot Chat ID, 라즈베리파이가 필요합니다.
  
  봇 토큰과 ID 양식:
 
 ### 출력용 봇
API_TOKEN = '{Token}'
BOT_CHAT_ID = '{chat id}'

### 입력용 봇
Tado_API_TOKEN = '{Token}'
Tado_BOT_CHAT_ID = "{chat id}"

참고:만약 그룹채팅에 봇을 작동시키고 싶다면 CHAT_ID 부분에 해당 그룹의 CHAT_ID를 넣으면 됩니다.
